# ⚫ Gomoku（五子棋）

## 遊戲說明

這是一款策略型棋類遊戲。玩家輪流在棋盤上放置黑白棋子，目標是先獲得五個連續的棋子（橫、豎或斜線）。

## 玩法規則

1. **目標**：先在棋盤上形成五個連續的棋子（同一顏色）
2. **棋盤**：標準 15×15 的棋盤
3. **輪流放置**：黑子先手，玩家輪流在空白位置放置棋子
4. **連線規則**：連線可以是：
   - 水平方向（橫向）
   - 垂直方向（縱向）
   - 斜線方向（左斜或右斜）
5. **遊戲結束**：任一方形成五子連線或棋盤滿局

## 控制方式

| 操作 | 方式 |
|------|------|
| 放置棋子 | 點擊棋盤上的空白位置 |
| 悔棋（如果支援） | 點擊悔棋按鈕 |
| 重新開始 | 點擊重新開始按鈕 |

## 遊戲特色

- 🎯 經典的策略棋類遊戲
- 👥 支援人機對戰或雙人遊戲
- 🧠 需要深度戰略思考
- 📋 簡單學習，深度無限

## 基本策略

1. **控制中心**：搶佔棋盤中心位置
2. **進攻與防守**：
   - 當對手快要形成連線時立即阻擋
   - 同時建立自己的進攻線路
3. **製造機會**：建立多個可能連線的位置，讓對手難以全部防守
4. **邊角佈局**：邊角的棋子限制較多，中心棋子更具彈性
5. **長期規劃**：提前規劃 3-5 步，不只看眼前

## 常見戰術

- **雙攻**：同時推進兩條線，迫使對手無法全部防守
- **包圍**：用多個棋子限制對手的活動空間
- **誘騙**：故意創造虛假威脅，引導對手錯誤防守

## 難度等級

- 初級：適合初學者
- 中級：具有基本戰略的 AI
- 高級：強大的 AI 對手
