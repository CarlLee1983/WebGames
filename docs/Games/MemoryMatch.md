# 🃏 Memory Match（記憶翻牌）

## 遊戲說明

測試您的記憶力！翻開卡牌找出相同的一對，所有卡牌配對後遊戲結束。

## 玩法規則

1. **目標**：在最少的翻牌次數內配對所有卡牌
2. **翻牌方式**：點擊卡牌翻開，如果兩張卡牌相同則配對成功
3. **配對成功**：成功配對的卡牌會保持翻開狀態
4. **配對失敗**：不符合的卡牌會自動翻回
5. **遊戲結束**：所有卡牌都配對成功

## 控制方式

| 操作 | 方式 |
|------|------|
| 翻開卡牌 | 點擊卡牌 |
| 重新開始 | 點擊重新開始按鈕 |

## 遊戲特色

- 🧠 訓練記憶力和觀察力
- 📊 計數系統追蹤翻牌次數
- 🎯 專注力測試遊戲

## 策略提示

1. 仔細觀察每張卡牌，記住它們的位置
2. 先翻開不同的卡牌以了解全局
3. 找出規律並建立心理地圖
4. 集中注意力，避免分心
5. 逐漸提高難度挑戰自己
