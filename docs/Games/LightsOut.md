# 💡 Lights Out（熄燈遊戲）

## 遊戲說明

經典的邏輯謎題遊戲。您需要熄滅所有的燈。點擊一個燈時，它和所有相鄰的燈都會切換狀態！

## 玩法規則

1. **目標**：熄滅網格中的所有燈
2. **點擊機制**：點擊一個燈時：
   - 該燈的狀態切換（開→關 或 關→開）
   - 上下左右相鄰的燈的狀態也會切換
   - 對角線的燈不受影響
3. **遊戲結束**：所有燈都熄滅時獲勝

## 控制方式

| 操作 | 方式 |
|------|------|
| 點擊燈 | 點擊網格中的任何燈 |
| 重新開始 | 點擊重新開始按鈕 |

## 遊戲特色

- 🧩 邏輯推理遊戲
- 🌐 相鄰燈光影響機制
- 💭 需要策略性思考

## 策略提示

1. **從邊緣開始**：先點擊邊緣和角落的燈
2. **分析影響**：每個點擊影響 5 個燈（中心 + 4 個方向）
3. **逆向思考**：如果某燈應該亮，倒推哪些燈應該被點擊
4. **記錄狀態**：追蹤每次點擊的效果
5. **多次嘗試**：有些謎題可能需要多次試驗才能找到答案

## 難度變化

遊戲會提供不同大小的網格和複雜度，測試您的邏輯推理能力。
