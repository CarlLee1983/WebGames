# 🔢 Sudoku（數獨）

## 遊戲說明

經典的邏輯數字謎題遊戲。根據簡單的規則填充 9×9 網格，訓練您的邏輯推理能力。

## 玩法規則

1. **目標**：用數字 1-9 填充整個 9×9 的網格
2. **基本規則**：
   - 每一行必須包含 1-9 的每個數字恰好一次
   - 每一列必須包含 1-9 的每個數字恰好一次
   - 每個 3×3 的小方格必須包含 1-9 的每個數字恰好一次
3. **初始狀態**：網格中已有一些預填的數字作為線索
4. **遊戲結束**：正確填充所有空白格子

## 控制方式

| 操作 | 方式 |
|------|------|
| 選擇格子 | 點擊網格中的任何空白格子 |
| 輸入數字 | 點擊數字按鈕或用鍵盤輸入 1-9 |
| 清除格子 | 按刪除鍵或點擊清除按鈕 |
| 撤銷操作 | 點擊撤銷按鈕 |
| 提示 | 點擊提示按鈕獲得幫助 |

| 專家 | 極具挑戰 | 少於 20 |

## 解題策略

### 基本技巧

1. **唯一候選數字**：如果一個格子只有一個可能的數字，填入它
2. **隱藏的單數**：如果某行/列/小方格中只有一個位置可以放置某個數字，填入它

### 高級技巧

3. **指向對**：在小方格中確認某對數字的位置限制
4. **盒線減少法**：如果某行在小方格內只能放置某數字
5. **X-Wing**：識別跨越多個單元的限制模式

## 使用提示

- 開始時先填入那些只有一個可能選項的格子
- 使用鉛筆標記每個格子的可能數字
- 專注於最受限制的行、列或小方格
- 不要猜測，邏輯推理才是解決方案
