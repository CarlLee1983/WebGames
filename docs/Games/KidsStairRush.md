# 🎯 Kids Stair Rush（兒童下樓衝刺）

## 遊戲說明

幫助小朋友急速衝下樓梯！躲避障礙物，反應敏捷，隨著樓梯加速，看你能走多遠！

## 玩法規則

1. **目標**：盡可能長地保持衝下樓梯，避免碰撞
2. **自動下滑**：小朋友會自動沿著樓梯下滑，您只需控制方向
3. **躲避障礙**：
   - 避免撞到樓梯上的障礙物
   - 障礙物會不同變動位置
4. **加速機制**：隨著時間推移，下滑速度逐漸加快
5. **遊戲結束**：撞到障礙物或摔下樓梯
6. **得分**：根據下滑的距離和時間計算分數

## 控制方式

| 操作 | 按鍵 |
|------|------|
| 向左移動 | <kbd>←</kbd> / <kbd>A</kbd> |
| 向右移動 | <kbd>→</kbd> / <kbd>D</kbd> |
| 暫停遊戲 | <kbd>Space</kbd> / <kbd>P</kbd> |

## 遊戲特色

- 👶 兒童友好的設計和角色
- 🎨 色彩鮮豔的視覺風格
- ⚡ 快節奏的動作遊戲
- 📈 難度逐漸遞增
- 🏆 成就和高分排行

## 難度階段

遊戲會經歷不同的難度階段：

| 階段 | 速度 | 障礙密度 |
|------|------|---------|
| 初期 | 緩慢 | 稀疏 |
| 中期 | 中等 | 中等 |
| 後期 | 快速 | 密集 |

## 策略提示

1. **預判走向**：提前規劃路線，不要臨時反應
2. **保持中心**：盡量在樓梯中間移動，擁有更多迴旋空間
3. **小幅移動**：避免大幅的橫向移動，細微調整更精準
4. **觀察模式**：障礙物可能有一定的出現規律，學習並預測
5. **保持警惕**：隨著速度加快，反應時間更短，保持集中注意力
6. **放鬆身心**：不要過度緊張，保持冷靜能提升表現

## 遊戲技巧

- 開始時不用急，適應遊戲節奏
- 速度加快後更要穩定，不要亂按鍵
- 利用樓梯邊緣作為參考點
- 熟悉後嘗試挑戰更高難度

## 健康提示

- 兒童應在監護人指導下遊戲
- 適度遊戲，避免過度疲勞
- 定期休息，保護眼睛
- 保持良好坐姿
