# 🧩 Tetris（俄羅斯方塊）

## 遊戲說明

堆疊下落的方塊，消除完整的行來獲得分數。不要讓方塊堆積到頂部！

## 玩法規則

1. **目標**：獲得最高分數，盡可能長時間存活
2. **方塊下落**：不同形狀的方塊不斷從頂部下落
3. **消除規則**：當一行完全被填滿時，該行消除並獲得分數
4. **加速機制**：隨著得分增加，方塊下落速度會加快
5. **遊戲結束**：方塊堆積到頂部，無法再放置新方塊

## 控制方式

| 操作 | 按鍵 |
|------|------|
| 向左移動 | <kbd>←</kbd> / <kbd>A</kbd> |
| 向右移動 | <kbd>→</kbd> / <kbd>D</kbd> |
| 旋轉方塊 | <kbd>↑</kbd> / <kbd>W</kbd> / <kbd>Space</kbd> |
| 快速下落 | <kbd>↓</kbd> / <kbd>S</kbd> |
| 暫停遊戲 | <kbd>P</kbd> |

## 遊戲特色

- 📈 分數和等級系統
- ⚡ 難度遞增的挑戰
- 🎮 經典的遊戲體驗
- 📊 下一個方塊預覽

## 策略提示

1. **計劃放置**：預視下一個方塊，提前規劃位置
2. **保持空隙**：在牆邊留一些空隙以增加靈活性
3. **早期堆積**：不要讓方塊堆太高
4. **消除多行**：盡量一次消除多行以獲得更多分數
5. **保持平穩**：避免形成尖峰和不規則的表面
6. **快速反應**：隨著速度加快，提高您的反應速度
