# 🌾 Farm（農場經營）

## 遊戲說明

一款放鬆的像素農場經營遊戲！種植莊稼、澆水、收穫並換取金幣和經驗值。打造您自己的夢想農場。

## 玩法規則

1. **目標**：經營農場、獲得金幣和經驗值，升級農場設施和物品
2. **種植莊稼**：
   - 選擇農地和作物類型
   - 點擊種植按鈕播種
   - 莊稼需要一定時間才能成熟
3. **澆水機制**：
   - 幼苗需要定期澆水才能健康成長
   - 澆水會加速莊稼的生長
   - 忘記澆水可能導致莊稼枯死
4. **收穫**：
   - 莊稼成熟後點擊收穫
   - 獲得該莊稼的資源和金幣
5. **升級系統**：
   - 獲得的金幣可用於升級農場設施
   - 經驗值用於提升農民等級
6. **金幣和資源**：
   - 收穫莊稼獲得金幣
   - 完成任務獲得獎勵
   - 使用金幣解鎖新作物和設施

## 控制方式

| 操作 | 方式 |
|------|------|
| 選擇農地 | 點擊農田 |
| 種植莊稼 | 點擊種植按鈕並選擇作物 |
| 澆水 | 點擊莊稼並選擇澆水 |
| 收穫莊稼 | 點擊成熟的莊稼 |
| 打開商店 | 點擊商店圖標 |
| 升級設施 | 在菜單中選擇升級 |

## 設施和升級

隨著進度提升，您可以解鎖和升級：

- 🏠 穀倉：儲存資源
- 💧 灌溉系統：自動澆水
- 🚜 拖拉機：加速收穫
- 🏪 市集攤位：出售產品

## 經營策略

1. **初期階段**：
   - 專注於種植快速生長的作物（小麥）
   - 積累資金用於升級

2. **中期階段**：
   - 種植高價值作物（番茄、玉米）
   - 升級灌溉系統以自動化澆水

3. **後期階段**：
   - 種植最有利潤的作物
   - 完全自動化農場運作
   - 收集稀有植物

## 提示

1. **定時澆水**：不要讓莊稼乾枯，定期檢查
2. **多樣化種植**：同時種多種作物，分散風險
3. **規劃佈局**：合理安排農地位置，便於管理
4. **升級優先級**：優先升級澆水系統，節省時間
5. **參與活動**：完成特殊任務獲得額外獎勵
6. **耐心經營**：農場建設需要時間，享受過程

## 放鬆遊戲特性
