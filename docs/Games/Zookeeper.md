# 🦁 Zookeeper（動物園管理員）

## 遊戲說明

連連看式的動物園謎題遊戲！將相同的動物連在一起，觸發重新洗牌，努力達成目標分數。

## 玩法規則

1. **目標**：在限定的時間或步數內達成目標分數
2. **連接動物**：
   - 點擊一個動物，然後點擊另一個相同的動物來連接它們
   - 必須能找到路徑連接（路徑不能穿過其他動物）
   - 只有相同種類的動物才能連接
3. **消除機制**：成功連接的動物對會從棋盤消失
4. **重新排列**：當無法再進行連接時，棋盤會自動重新排列
5. **分數計算**：
   - 連接動物對獲得分數
   - 連接難度越高，獲得的分數越多
   - 完成挑戰目標賺取額外獎勵

## 控制方式

| 操作 | 方式 |
|------|------|
| 選擇動物 | 點擊第一個動物 |
| 連接動物 | 點擊第二個相同的動物 |
| 重新開始 | 點擊重新開始按鈕 |

## 遊戲特色

- 🦁 可愛的動物角色和設計
- 🎯 目標導向的遊戲玩法
- 🌟 分數和獎勵系統
- 📈 逐級增加難度的挑戰

## 動物角色

遊戲中會出現各種有趣的動物，每種都有獨特的設計，增加遊戲的視覺樂趣。

## 策略提示

1. **優先消除容易連接的配對**，保持棋盤流暢
2. **規劃路徑**：尋找最短路徑來連接動物
3. **創造空間**：消除動物後會產生新的連接機會
4. **利用重新排列**：當棋盤卡住時，重新排列通常會產生新的機會
5. **管理移動**：每個操作都很重要，計劃好您的連接
6. **觀察全局**：掃描整個棋盤尋找最佳的連接路徑

## 遊戲難度

難度會隨著進度增加，帶來更多的挑戰和獎勵。每個關卡都有特定的目標分數和時間限制。

## 提示和幫助

如果您卡住了，可以：
- 重新排列棋盤以獲得新的機會
- 查看可用的動物配對
- 使用特殊技能或道具（如果可用）
