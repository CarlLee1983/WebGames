# 🐍 Snake（貪食蛇）

## 遊戲說明

經典的貪食蛇遊戲。控制蛇吃掉食物並不斷成長，同時避免撞到牆壁或咬到自己的身體。

## 玩法規則

1. **目標**：吃掉盡可能多的食物，獲得最高分數
2. **食物出現**：每吃掉一個食物，蛇的身體增加一節，新的食物會隨機出現
3. **遊戲結束條件**：
   - 蛇頭撞到遊戲邊界的牆壁
   - 蛇頭撞到自己的身體

## 控制方式

| 操作 | 按鍵 |
|------|------|
| 向上移動 | <kbd>↑</kbd> / <kbd>W</kbd> |
| 向下移動 | <kbd>↓</kbd> / <kbd>S</kbd> |
| 向左移動 | <kbd>←</kbd> / <kbd>A</kbd> |
| 向右移動 | <kbd>→</kbd> / <kbd>D</kbd> |

## 遊戲特色

- 🎮 簡單易學，難以精通
- 📈 分數系統記錄您的最佳成績
- ⚡ 難度持續提升的遊戲體驗

## 策略提示

1. 不要急著改變方向，計劃好路線
2. 留意蛇的身體位置，避免誤踩
3. 靠近邊界時更要小心謹慎
4. 養成規律的移動習慣提升反應速度
